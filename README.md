### Prerequisites

In order to run this project you need:

> - Git
> - IDE(e.g.: VsCode, Sublime,...)
> Proficiency in the use of CSS(Flexbox)
> Advance HTML
> DOM proficiency with Javascript

Clone this repository to your desired folder:

